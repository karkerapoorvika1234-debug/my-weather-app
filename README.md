# My-weather-app
📌 Overview

The Weather App is a lightweight web application that provides current weather updates for any city. It retrieves live weather information through an API and presents it in a clean, easy-to-use interface.
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
✨ Features

🌍 Get weather updates for cities worldwide

🌡️ View current temperature instantly

💨 Check wind speed and humidity

☁️ Display weather condition with matching icons

📱 Mobile-friendly and responsive layout

⚠️ Handles invalid city searches gracefully

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

HTML

CSS

JavaScript

OpenWeatherMap API

Fetch API
